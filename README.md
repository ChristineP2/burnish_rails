## Installation
Add this line to your application's Gemfile:

```ruby
gem "burnish_rails"
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install burnish_rails
```

Create Default Config File:
```bash
$ rails generate burnish_rails:install 
```
